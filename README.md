# risk-os
Um sistema de terminal, que substitui quase completamente o termux, deixando apenas os comandos que já vem instalado.

# install
Os comandos para instalação so funciona em um terminal termux com ohmyzsh/zsh.

<table>
<tr><th>Method</th><th>Command</th></tr>
<tr><td><b>curl</b></td><td><code>sh -c "$(curl -fsSL https://raw.githubusercontent.com/sichvs/risk-os/main/build.bash)"</code></td></tr>
<tr><td><b>wget</b></td><td><code>sh -c "$(wget -O- https://raw.githubusercontent.com/sichvs/risk-os/main/build.bash)"</code></td></tr>
</table>
