# risk-os
Um sistema de terminal, que substitui quase completamente o termux, deixando apenas os comandos que já vem instalado.

# install
Os comandos para instalação so funciona em um terminal termux com ohmyzsh/zsh.

**curl**
```bash # sh -c "$(curl -fsSL https://raw.githubusercontent.com/sichvs/risk-os/main/acct/build.bash)"```
**wget**
```bash # sh -c "$(wget -q0- https://raw.githubusercontent.com/sichvs/risk-os/main/acct/build.bash```
